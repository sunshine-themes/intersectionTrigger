## Signature

```ts
new IntersectionTrigger(options?: IntersectionTriggerOptions)
```

Creates a new IntersectionTrigger instance. The `options` argument is optional; all properties have sensible defaults.

## Options

| Property | Type | Default | Description |
|----------|------|---------|-------------|
| `defaults` | [`TriggerOptions`](/guide/methods#trigger-options) | `{}` | Default properties inherited by every trigger added via `add()`. |
| `root` | `HTMLElement \| string \| null` | `null` | The scrollable ancestor element to use as the intersection root. Pass a CSS selector string, an element reference, or `null` for the viewport. |
| `rootEnter` | `RootPosition` | `'100%'` | The position on the root boundary that constitutes an "enter" intersection. Accepts a percentage or pixel string (e.g., `'75%'`, `'200px'`), or a function returning one. |
| `rootLeave` | `RootPosition` | `'0%'` | The position on the root boundary that constitutes a "leave" intersection. Same value format as `rootEnter`. |
| `axis` | `'y' \| 'x'` | `'y'` | The scroll axis to observe. Use `'x'` for horizontal scrolling containers. |
| `name` | `string` | `''` | An optional name for the instance. Displayed in Guides overlays for debugging. |
| `guides` | `boolean \| GuidesOptions` | `false` | Enable visual debugging overlays. Pass `true` for defaults, or an object to customize colors and labels. |
| `onScroll` | `EventHandler` | `undefined` | A callback fired on every scroll event within the root element. Receives the native `Event` object. |

## Examples

### Basic Constructor

Uses the viewport as the root with all defaults.

```js
import IntersectionTrigger from 'intersectiontrigger';

const it = new IntersectionTrigger();
```

### Custom Root Element

Observe intersections within a specific scrollable container instead of the viewport.

```js
const it = new IntersectionTrigger({
	root: '#scroll-container',
});
```

You can also pass an element reference directly:

```js
const container = document.querySelector('#scroll-container');

const it = new IntersectionTrigger({
	root: container,
});
```

### Horizontal Axis

Track intersections along the horizontal scroll axis.

```js
const it = new IntersectionTrigger({
	root: '#horizontal-scroll',
	axis: 'x',
	rootEnter: '100%',
	rootLeave: '0%',
});
```

### With Guides Enabled

Show visual debugging overlays that mark the enter and leave positions on both the root and trigger elements.

```js
const it = new IntersectionTrigger({
	root: '#scroll-container',
	name: 'My Instance',
	guides: true,
});
```

Customize guide colors and labels:

```js
const it = new IntersectionTrigger({
	root: '#scroll-container',
	guides: {
		enter: {
			trigger: { backgroundColor: 'green', text: 'Trigger Enter' },
			root: { backgroundColor: 'green', text: 'Root Enter' },
		},
		leave: {
			trigger: { backgroundColor: 'red', text: 'Trigger Leave' },
			root: { backgroundColor: 'red', text: 'Root Leave' },
		},
	},
});
```

### With Default Trigger Options

Set callbacks or positions that apply to every trigger added to this instance.

```js
const it = new IntersectionTrigger({
	root: '#scroll-container',
	defaults: {
		enter: '25%',
		leave: '75%',
		once: true,
		onEnter: (trigger) => trigger.classList.add('visible'),
	},
});

it.add('.card');
```
