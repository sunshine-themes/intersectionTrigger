## Why IntersectionTrigger?

A zero-dependency TypeScript library that enhances the IntersectionObserver API — resolving the "target height > root height" limitation and unlocking scroll-based animations with a plugin architecture.

<div className="feature-grid" style={{ margin: '2rem 0' }}>
  <FeatureCard icon="⚡" title="Zero Dependencies" description="No external libraries required. Tiny footprint, blazing fast." />
  <FeatureCard icon="🚀" title="Performance Optimized" description="Built on IntersectionObserver — zero scroll events on the main thread." />
  <FeatureCard icon="🎬" title="Controlled Animations" description="Play, pause, resume, reverse, restart animations on enter/leave." />
  <FeatureCard icon="🔗" title="Linked Animations" description="Link any animation to the scrollbar with configurable smoothing." />
  <FeatureCard icon="🎯" title="Snapping System" description="Snap scroll-linked animations to defined points." />
  <FeatureCard icon="🧩" title="Plugin Architecture" description="Extend core with Animation, ToggleClass, and Guides plugins." />
  <FeatureCard icon="👁" title="Custom Roots" description="Use any element as the intersection root, not just the viewport." />
  <FeatureCard icon="📐" title="Visual Guides" description="Debug with visual overlays showing trigger/root positions." />
  <FeatureCard icon="↔" title="Axis Support" description="Vertical and horizontal intersection detection." />
  <FeatureCard icon="🎨" title="ToggleClass System" description="Rich CSS class toggling with granular control over actions." />
  <FeatureCard icon="📏" title="Resize Observation" description="Automatic recalculation when the root resizes." />
