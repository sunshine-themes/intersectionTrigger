---
title: IntersectionTrigger Class
description: Full API reference for the IntersectionTrigger class
---

The `IntersectionTrigger` class is the main entry point for the library. It wraps the native `IntersectionObserver` with enhanced positioning logic that handles cases where the trigger element is taller than the root.

## Constructor

```ts
new IntersectionTrigger(options?: IntersectionTriggerOptions)
```

Creates a new IntersectionTrigger instance with the given configuration. See [IntersectionTriggerOptions](/api/type-definitions) for the full options type.

```js
const it = new IntersectionTrigger({
  rootEnter: '100%',
  rootLeave: '0%',
  axis: 'y',
  root: null,
  guides: false,
  defaults: {
    enter: '0%',
    leave: '100%',
    once: false
  }
});
```

## Instance Properties

| Property | Type | Description |
| -------- | ---- | ----------- |
| `id` | `number` | Unique integer assigned to each instance (auto-incremented) |
| `triggers` | `HTMLElement[]` | Array of all trigger elements added to the instance |
| `observer` | `IntersectionObserver \| undefined` | The internal IntersectionObserver; `undefined` after disconnect |
| `rootBounds` | `DOMRectReadOnly` | The current root size and position |
| `killed` | `boolean` | Whether the instance has been killed |
| `axis` | `string` | Scroll axis (`'y'` or `'x'`) |
| `name` | `string \| undefined` | Optional name for the instance (used in guides labels) |

### Plugin Properties

These properties are set when the corresponding plugin is activated:

| Property | Type | Description |
| -------- | ---- | ----------- |
| `toggleClass` | `ToggleClass \| undefined` | ToggleClass plugin instance |
| `animation` | `Animation \| undefined` | Animation plugin instance |
| `guides` | `Guides \| undefined` | Guides plugin instance |

## Instance Methods

### `add()`

```ts
add(trigger: Trigger, options?: TriggerOptions): this
```

Adds one or more trigger elements to be observed. `trigger` accepts a CSS selector string, an `HTMLElement`, an array of elements, or a `NodeListOf<HTMLElement>`. Returns the instance for chaining.

```js
it.add('.box');
it.add(document.querySelector('.hero'), { enter: '25%', leave: '75%' });
it.add([el1, el2, el3], { once: true, onEnter: cb });
```

See [TriggerOptions](/api/trigger-options) for all available options.

### `remove()`

```ts
remove(trigger: Trigger): this
```

Removes trigger elements from observation. Accepts the same `Trigger` types as `add()`. Returns the instance for chaining.

```js
it.remove('.box');
it.remove(element);
```

### `update()`

```ts
update(): void
```

Disconnects and recreates the internal `IntersectionObserver`, then re-observes all stored triggers. Use this after changing trigger positions or when the layout changes. Updates guides if active.

### `kill()`

```ts
kill(): void
```

Permanently destroys the instance: disconnects the observer, removes all event listeners (scroll, resize), kills all active plugins, clears all triggers, and removes the instance from the global registry. After calling `kill()`, the instance cannot be reused.

## Static Methods

### `getInstances()`

```ts
static getInstances(): IntersectionTrigger[]
```

Returns an array of all active IntersectionTrigger instances.

### `getInstanceById()`

```ts
static getInstanceById(id: number): IntersectionTrigger | undefined
```

Returns the instance with the given `id`, or `undefined` if not found.

### `registerPlugins()`

```ts
static registerPlugins(plugins: Plugin[]): number
```

Registers one or more plugins. Must be called before creating instances that use plugin features. Returns the new length of the registered plugins array.

```js
IntersectionTrigger.registerPlugins([Animation, ToggleClass, Guides]);
```

### `getRegisteredPlugins()`

```ts
static getRegisteredPlugins(): Plugin[]
```

Returns the array of currently registered plugin classes.

### `update()`

```ts
static update(): void
```

Calls `update()` on every active instance.

### `kill()`

```ts
static kill(): void
```

Kills all active instances. Iterates through and destroys each one.
