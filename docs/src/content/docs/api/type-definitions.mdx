title: Type Definitions
description: TypeScript types and interfaces
---

IntersectionTrigger is written in TypeScript and ships with full type definitions -- no additional `@types` package is needed. All types are exported from the main entry point.

## Core Types

### `Trigger`

```ts
type Trigger = string | HTMLElement | HTMLElement[] | NodeListOf<HTMLElement>;
```

A flexible element selector used throughout the API. Accepts a CSS selector string, a single DOM element, an array of elements, or a NodeList.

### `Root`

```ts
type Root = HTMLElement | null;
```

The root element for observation. `null` defaults to the browser viewport.

### `EventHandler`

```ts
type EventHandler = (event: Event) => void;
```

A generic event handler function.

### `ItCallbackFunction`

```ts
type ItCallbackFunction = (trigger: HTMLElement, it: IntersectionTrigger) => void;
```

Callback function for intersection events. Receives the trigger element and the IntersectionTrigger instance.

### `RootValue`

```ts
type RootValue = `${number}${'%' | 'px'}`;
```

A position value for the root, expressed as a percentage or pixel string (e.g., `'100%'`, `'200px'`).

### `TriggerValue`

```ts
type TriggerValue = `${number}%`;
```

A position value for a trigger element, expressed as a percentage string (e.g., `'25%'`, `'0%'`).

### `RootPosition`

```ts
type RootPosition = RootValue | ((it?: IntersectionTrigger) => RootValue);
```

A root position: either a static value string or a function that returns one, receiving the IntersectionTrigger instance.

### `TriggerPosition`

```ts
type TriggerPosition = TriggerValue | ((it?: IntersectionTrigger) => TriggerValue);
```

A trigger position: either a static percentage string or a function that returns one.

## Interface Types

### `IntersectionTriggerOptions`

Configuration for the `IntersectionTrigger` constructor. See the [IntersectionTrigger Class](/api/intersectiontrigger-class) page.

```ts
interface IntersectionTriggerOptions {
  defaults?: TriggerOptions;
  rootEnter?: RootPosition;
  rootLeave?: RootPosition;
  axis?: string;
  name?: string;
  root?: Root | string;
  guides?: boolean | GuidesOptions;
  onScroll?: EventHandler;
}
```

### `TriggerOptions`

Configuration passed to `add()`. See the [TriggerOptions](/api/trigger-options) page.

```ts
interface TriggerOptions {
  enter?: TriggerPosition;
  leave?: TriggerPosition;
  once?: boolean;
  onEnter?: ItCallbackFunction;
  onLeave?: ItCallbackFunction;
  onEnterBack?: ItCallbackFunction;
  onLeaveBack?: ItCallbackFunction;
  toggleClass?: string | ToggleClassOptions[];
  animation?: Anime<AnimeInstance> | AnimationOptions;
}
```

## Plugin Types

### Animation Types

See the [Animation Plugin](/plugins/animation) page for usage details.

```ts
interface AnimationOptions {
  instance: Anime<AnimeInstance>;
  toggleActions?: `${AnimationToggleActions} ${AnimationToggleActions} ${AnimationToggleActions} ${AnimationToggleActions}`;
  link?: number | boolean;
  snap?: SnapConfiguration;
}
```

```ts
type AnimationToggleActions =
  | 'none' | 'play' | 'resume' | 'restart' | 'reset'
  | 'pause' | 'complete' | 'reverse' | 'kill';
```

```ts
interface SnapOptions {
  to: number | number[] | string;
  after?: number;
  speed?: number;
  maxDistance?: number;
  onStart?(it: IntersectionTrigger): void;
  onComplete?(it: IntersectionTrigger): void;
}
```

```ts
type SnapConfiguration = SnapOptions | boolean | number | number[];
```

```ts
type AnimeInstance = anime.AnimeInstance | anime.AnimeTimelineInstance;
```

### ToggleClass Types

See the [ToggleClass Plugin](/plugins/toggleclass) page for usage details.

```ts
interface ToggleClassOptions {
  targets?: Trigger;
  toggleActions?: `${ClassToggleActions} ${ClassToggleActions} ${ClassToggleActions} ${ClassToggleActions}`;
  classNames: string;
}
```

```ts
type ClassToggleActions = 'add' | 'remove' | 'none';
```

### Guides Types

See the [Guides Plugin](/plugins/guides-plugin) page for usage details.

```ts
interface GuidesOptions {
  enter?: GuideElements;
  leave?: GuideElements;
}
```

```ts
interface GuideElements {
  trigger?: GuideProperties;
  root?: GuideProperties;
}
```

```ts
interface GuideProperties {
  backgroundColor?: string;
  color?: string;
  text?: string;
}
```

## Plugin Interface

```ts
type Plugin = typeof Animation | typeof ToggleClass | typeof Guides;
```

```ts
interface IntersectionTriggerPlugin {
  name: string;
}
```

Each plugin class has a static `pluginName` property that identifies it (`'animation'`, `'toggleClass'`, or `'guides'`).
