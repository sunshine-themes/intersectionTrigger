Install IntersectionTrigger using your preferred package manager, load it from a CDN, or download the files directly.

## Package Manager

Install from the npm registry:

:::code-group

```bash [npm]
npm install intersectiontrigger
```

```bash [Yarn]
yarn add intersectiontrigger
```

:::

## Import Methods

IntersectionTrigger ships multiple import targets depending on which features you need.

### Core Only

Import the core library with no plugins:

```js
import IntersectionTrigger from 'intersectiontrigger';
```

This gives you the base `IntersectionTrigger` class with intersection-based triggers, callbacks, and custom root support.

### With Plugins

Import the core and register individual plugins:

```js
import IntersectionTrigger from 'intersectiontrigger';
import Animation from 'intersectiontrigger/animation';
import ToggleClass from 'intersectiontrigger/toggleclass';
import Guides from 'intersectiontrigger/guides';

IntersectionTrigger.registerPlugins([Animation, ToggleClass, Guides]);
```

Register only the plugins you need to keep your bundle lean.

### Bundle Import

Import everything (core + all plugins) in a single statement:

```js
import IntersectionTrigger from 'intersectiontrigger/bundle';
```

The bundle automatically registers the Animation, ToggleClass, and Guides plugins.

## CDN

Use IntersectionTrigger directly in the browser without a build step. All CDN builds are hosted on [jsDelivr](https://www.jsdelivr.com/package/npm/intersectiontrigger).

### Core (IIFE / Browser Global)

```html
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/core/core.browser.min.js"></script>
<script>
  var it = new it_core({
    default: { enter: '20%', leave: '80%' }
  });
  it.add('.box');
</script>
```

### Bundle (IIFE / Browser Global)

```html
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/intersectiontrigger-bundle.browser.js"></script>
<script>
  var it = new intersectiontrigger({
    default: { enter: '20%', leave: '80%' }
  });
  it.add('.box');
</script>
```

### Bundle (ES Module)

```html
<script type="module">
  import IntersectionTrigger from 'https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/intersectiontrigger-bundle.browser.mjs';

  const it = new IntersectionTrigger({
    defaults: { enter: '20%', leave: '80%' }
  });
  it.add('.box');
</script>
```

### Per-Plugin CDN Files

Load only the plugins you need alongside the core:

```html
<!-- Core -->
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/core/core.browser.min.js"></script>

<!-- Load individual plugins -->
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/plugins/animation/animation.browser.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/plugins/toggleclass/toggleclass.browser.min.js"></script>
<script src="https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/plugins/guides/guides.browser.min.js"></script>

<script>
  it_core.registerPlugins([it_animation, it_toggleClass, it_guides]);
</script>
```

## Browser Globals

When loaded via `<script>` tags (IIFE builds), IntersectionTrigger exposes the following globals:

| Global                  | Source    | Description                                  |
| ----------------------- | --------- | -------------------------------------------- |
| `it_core`               | Core      | The `IntersectionTrigger` class              |
| `it_animation`          | Animation | The Animation plugin class                   |
| `it_toggleClass`        | ToggleClass | The ToggleClass plugin class               |
| `it_guides`             | Guides    | The Guides plugin class                      |
| `intersectiontrigger`   | Bundle    | The `IntersectionTrigger` class with all plugins pre-registered |

## Download

Download the latest release from [GitHub Releases](https://github.com/sunshine-themes/intersectionTrigger/releases) and include the files in your project.

## Hello World

Here is the smallest working example to get you started:

```html
<div class="box" style="height: 200px; width: 200px; background: steelblue;">
  I will be observed!
</div>

<script type="module">
  import IntersectionTrigger from 'https://cdn.jsdelivr.net/npm/intersectiontrigger@latest/intersectiontrigger-bundle.browser.mjs';

  const it = new IntersectionTrigger({
    defaults: { enter: '20%', leave: '80%' }
  });

  it.add('.box', {
    onEnter(trigger) {
      trigger.style.background = 'tomato';
      console.log('Entered!', trigger);
    },
    onLeave(trigger) {
      trigger.style.background = 'steelblue';
      console.log('Left!', trigger);
    }
  });
</script>
```

Scroll the page so `.box` crosses the 20% / 80% thresholds to see the callbacks fire.
